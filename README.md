Smart Fire Alarm System (Arduino) 🚨
An electronic fire alarm system designed to provide early warning in case of smoke or fire, simulated using TinkerCAD.

🎯 Project Goal
To create an affordable and efficient warning system for small spaces like rooms or offices, using basic electronic components.

🛠️ Components Used:

* Arduino UNO: Microcontroller for data processing.


* MQ-2 Gas Sensor: Detects smoke and gas levels.


* LM35 Temperature Sensor: Monitors ambient temperature.


* Actuators: Piezo Buzzer (sound) and LED (light) for alerts.

⚙️ How It Works
The MQ-2 sensor constantly monitors air quality.

If smoke or gas levels exceed a predefined threshold, a signal is sent to the Arduino.

The system triggers a sound alarm and a flashing LED until the environment is safe.
